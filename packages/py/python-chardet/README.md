For the forseeable future, this package is pinned to the `6.0.0.post1` version. The `7.x` versions are a full rewrite using Claude Code, and re-licensed[^1][^2]. While the licensing and AI issue is one thing, the versions so far (latest `7.4.3`) appear to break other programs and test suites. Later versions should be evauluated for further fixes and breakages.

Both Arch and Fedora do not appear to be updating to the `7.x` version series.
